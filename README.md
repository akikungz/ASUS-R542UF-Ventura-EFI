## Spec
- CPU: Intel Core i3 8130U (Kaby Lake Refresh)
- Ram: 12GB 2400MHz (8 + 4)
- iGPU: Intel UHD 620

- LAN: Realtek GbE

## Useable Function
- LAN + Wifi + Bluetooth

- 3.5mm Combo (not auto switch/restart to work)
- Sleep (Auto wake but sleep work on inactive)
